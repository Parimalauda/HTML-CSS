# Portfolio Card

A sleek, responsive portfolio card built with HTML and CSS to showcase skills, projects, and contact information in a clean and modern design.

## Features

- Responsive design for mobile, tablet, and desktop
- Clean and minimal UI with smooth hover effects
- Includes sections for profile photo, bio, skills, projects, and contact links
- Easy to customize and extend with your own details
